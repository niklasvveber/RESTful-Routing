RESTful Routing

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose
Index /entities GET List of entities
New /entities/new GET Show new entity form
Create /entities POST Create a new entity, the redirect somewhere
Show /entities/:id GET Show info about one specific entity
Edit /entities/:id/edit GET Show edit form for one entity
Update /entities/:id PUT Update a particular entity, then redirect somewhere
Destroy /entities/:id DELETE Delete a particular entity, then redirect somewhere
